Entry Data & Verifikasi Gambar Produk
Tentang Pekerjaan

Kamu akan bertugas untuk memasukkan dan memverifikasi data produk digital untuk katalog marketplace kami.

Berikut tugas utamanya:

 

  1. Menerima 100 file gambar produk (.jpg/.png) dari kami melalui dashboard.
  2. Memeriksa kelengkapan dan kualitas gambar (apakah resolusi cukup, tidak blur, dan tidak mengandung watermark).
  3. Memasukkan nama produk, kategori, dan deskripsi singkat ke template Excel yang sudah disiapkan.
  4. Menambahkan 3 kata kunci (keywords) relevan untuk setiap produk.
  5. Upload kembali file Excel dan beri checklist gambar yang lolos verifikasi.
Selengkapnya
Kualifikasi
  1. Teliti dan rapi dalam input data
  2. Bisa menggunakan Microsoft Excel atau Google Sheets
  3. Memiliki koneksi internet stabil
  4. Mampu menyelesaikan dalam waktu maksimal 5 hari
